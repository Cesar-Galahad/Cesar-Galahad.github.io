Planteamiento del problema

En la matriz de la aseguradora llegan 100 solicitudes diariamente la cual recibe diversos servicios como seguro de vida, de desempleo, de automóvil, etc. La aseguradora a presentado las siguientes situaciones:
El proceso es manual, lo que causa retrasos y complicaciones en la atención.
Las solicitudes llegan desde las sucursales a la matriz para su registro.
Los agentes no pueden ver el estado del proceso, lo que genera errores al ofrecer seguros.

Solución Ofimática

Nuestra propuesta para la problemática de NexoSeguros (Tu nexo con la seguridad), es realizar un sistema para la automatización de entrada, donde se pueda visualizar mediante una base de datos, las solicitudes, clientes registrados y realice una comparación para verificar y evitar que los agentes de seguros no registren un mismo cliente. También se realiza el programa para una mejor gestión de estos datos, donde se muestran los servicios que ofrecen como aseguradora y por último, la creación de una red para la conexión de nuestra matriz con la sucursal

Solución de Redes

Implementación de una infraestructura de red que garantice la conectividad segura entre sucursales y matriz.

Presupuesto

Se presenta un desglose detallado del costo de desarrollo de software, hardware necesario y servicios de red.
